# Swing starter project

This is the starter project for Swing desktop application.

Dependencies:

* FlatLaf: Flat look and feel, awesome project to instantly have nice UI (used by previous version of IntellijIDEA). Visit their project: https://www.formdev.com/flatlaf/
* MigLayout: Awesome project to layout components on the swing UI. Their domain is currently having some issue with ssl, proceed with your own risk or visit Github https://github.com/mikaelgrev/miglayout
